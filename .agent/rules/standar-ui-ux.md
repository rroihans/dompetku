# Standar Visual & UX
- **Mobile-First:** Aplikasi harus 100% responsif. Gunakan Bottom Navigation untuk mobile.
- **Tema:** Support Dark Mode & Light Mode (Toggle wajib ada).
- **Palet Warna:** Slate (900/950) dan Emerald (500/600).
- **Komponen:** Gunakan Shadcn UI dengan Tooltip untuk istilah akuntansi teknis.
- **Aset:** Gunakan Lucide React (ikon) dan CDN publik untuk aset eksternal.
